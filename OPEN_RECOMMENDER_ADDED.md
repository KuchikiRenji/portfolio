## ✅ Project Details

**Title:** Open Recommender — Open Source AI Recommendation System for Videos & Articles

**Slug:** `open-recommender`

**Period:** December 2023 - February 2024

**Impact:** "Self-hosted AI that learns your interests for timeless content"

**Description:**
Open source AI-powered recommendation system for YouTube videos and articles. Uses LLMs to learn from your interests and surface timeless, learning-focused content. Self-hosted solution built with TypeScript and Python, featuring tRPC for type-safe APIs and Prisma for database management.

## Tech Stack

**Backend:**
- Node.js
- tRPC (type-safe APIs)
- Prisma (ORM)
- Python (workers)

**Frontend:**
- React
- TypeScript

**AI/LLM:**
- Modelfusion
- LangChain
- OpenAI-compatible APIs

**Data Sources:**
- YouTube (via yt-dlp)
- Twitter/X (optional)
- Web search

**Tools:**
- Git/GitHub

## Technical Highlights

1. Backend built with Node.js, tRPC for end-to-end type safety, and Prisma ORM for database operations
2. Python workers for AI processing with Modelfusion, LangChain, and OpenAI-compatible APIs
3. React + TypeScript frontend with seamless integration to type-safe backend APIs
4. Content ingestion from YouTube (via yt-dlp), Twitter/X (optional), and web search

## Challenges & Solutions

1. **Learning user preferences** → Implemented LLM-based interest modeling that adapts to user behavior over time
2. **Content quality filtering** → Built AI pipeline to prioritize timeless, educational content over viral trends
