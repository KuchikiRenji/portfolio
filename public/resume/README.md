# Resume Folder

## How to Add Your Resume

1. **Save your resume PDF** with the filename: `Kuchiki_Renji_Resume.pdf`
2. **Place it in this folder**: `public/resume/`
3. The resume will be automatically available at: `/resume/Kuchiki_Renji_Resume.pdf`

## Recommended Resume Filename

- `Kuchiki_Renji_Resume.pdf` (primary)
- Alternative: `resume.pdf`

## File Requirements

- **Format**: PDF only
- **Size**: Recommended < 2MB for fast loading
- **Naming**: Use underscores or hyphens, no spaces

## After Adding Your Resume

The resume will be accessible via:

- Direct link: `https://your-domain.com/resume/Kuchiki_Renji_Resume.pdf`
- Download button in the About section
- Navbar "Resume" link (if added)

## Tips

- Keep filename consistent for easy updates
- Optimize PDF size before uploading
- Test the download link after deployment
